OVERVIEW

The STEAM (Science, Technology, Engineering, Art, and Math) Discovery Lab is one of Tufts Center for Engineering Education and Outreach’s (CEEO) research projects. The goal of the STEAM Discovery Lab is to provide educators with new and exciting ideas and resources for conducting hands-on activities in their classroom. The website provides teachers with inspiring lesson plans and activity ideas that they can tweak to best meet the needs of their students and subject area. The site also features activity ideas that combine different technologies in unconventional ways.

My goal as a UX designer was to create a user-friendly website by making the content discoverable and accessible as well as visually appealing.

INTERFACE DESIGN

The Draw2Code kit consists of two major components: paper coding blocks and a mobile application. Children can author their own animations by manipulating the paper coding blocks and execute the code with a mobile device. We designed elements of the interface to make both of tangible and virtual interfaces appropriate for young children. For example, no text is used, buttons are big enough, and icons are straightforward to understand. We also paid attention to involving both boys and girls, such as using colors to which both genders can feel a sense of belonging.


The Sprite block has an empty canvas area where children can draw their own character or place a small craft on it to use it as a character. The Event block is for triggering the virtual character to animate. The current version can use hand gestures such as rock paper scissors and graphical play button as event blocks. Each event block represents a particular command to trigger the program and has a unique visual depending on its type. The Action block has a grid where children can fill color in it with a different size and location that guide the virtual character’s appearance in each frame. A sequence of action blocks creates an animation.

The child can use the scan mode in the mobile application to load the sequence of the blocks as an animation and see the current state of the animation in the run mode.

IMPLEMENTATION

We fabricated Draw2Code paper coding blocks by printing them in colored paper and cutting out the paper. Draw2Code software is a web-based application written in Javascript. We used the p5.js library for building user interface components. We leveraged Teachable Machine to create image classification models to detect different coding blocks and hand gestures from images. We made image processing algorithms to parse the block images to data and control variables using computer vision techniques. We used a camera embedded in a mobile device to capture the block images.
