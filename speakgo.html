Speak Go

NOV 2017 - DEC 2017

OVERVIEW

Speak Go is a speech game that enhances memorizing with playful and engaging learning. We implemented Web Speech API to make a voice controlled interface.

MY ACTIVITIES

• Game Design
• Data Structures
• UI Design
• 2D Graphic Design & Animation
• Programming

CONTEXT

CS86: Object-Oriendted Programming of Graphical User Interface, Tufts University
Team: Kam Chan Kang

HOW IT WORKS?

Speak Go, which is easily expandable, is essentially a memorization game, that has several geometric shapes advance toward the user, requiring the user to say the name of the shape out loud in order to destroy it.

WHY IT IS WORTHY?

Our target user pool is young children, who struggle with learning geometric shapes. With the sense of urgency that the animation induces, it motivates the user to get the names of the shapes right in an entertaining way.

This program can be expanded with different objects (e.g., countries flags instead of shapes) and also in another language (due to Web Speech API's capabilities).

*It works on Chrome, desktop.
