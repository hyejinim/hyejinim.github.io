A Goal-Oriented Morning Alarm Clock

Personal Project | Capstone Project for Interaction Design Specialization
Sep 2016 - Nov 2016


My Role: User Research, UI/UX Design, Prototyping, User Testing

OVERVIEW

I designed a morning alarm clock app for people who struggle to wake up at a sceduled time. Instead of setting a single time to wake up as other alarm clocks do, my prototype provides two different times to set up, one for waking up and another for doing following task (e.g., go to work). The app focuses on achieving a high-level foal as waking up on time is often only a part of reaching the goal. That is, it emphasizes the time gap between wake-up time and task time to encourage a user to be more aware of the goal when setting the times and hitting the snooze button.
