Vismorn

SEP 2016 - NOV 2016

OVERVIEW

Vismorn is a goal-oriented morning alarm clock mobile application.

MY ACTIVITIES

• User Research/Testing
• UI/UX Design
• Prototyping

CONTEXT

Capstone Project, Interaction Design Specialization, UC San Diego on Coursera

I designed Vismorn for people who struggle to wake up at a sceduled time. Instead of setting a single time to wake up as other alarm clocks do, Vismorn provides two different times to set up, one for waking up and another for doing following task (e.g., go to work).

The app focuses on achieving a high-level goal as waking up on time is often only a part of reaching the goal. That is, it emphasizes the time gap between wake-up time and task time to encourage a user to be more aware of the goal when setting the times and hitting the snooze button.
