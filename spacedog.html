Laika Back To The Earth

SEP 2017 - OCT 2017

OVERVIEW

This is a 2D simulation game using Java. It has a game design mode and a play mode.

MY ACTIVITIES

• Game design
• User Interface Design
• Programming

CONTEXT

CS86: Object-Oriendted Programming of Graphical User Interface, Tufts University

BACKGROUND

This project was inspired by the real story of Laika, the Soviet stray dog who was sacrified for a space experiment and could not come back to the earth.

In this game, a player saves other dogs in the space and makes Laika come back home. I hope more peoplae are aware of the behind story of this game: animals' sacrifice and contribution.

I also would like to give a positive lesson to players through this game. It does not require fighting and competeting against enemies but helping ohters to achieve the success.
